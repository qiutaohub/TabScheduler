# TabScheduler

TabScheduler 是一个轻量级的 Chrome 浏览器扩展，旨在帮助用户更高效地管理和调度标签页。

## 特性

- 🕒 **定时任务**: 为标签页设置定时任务。
- 📂 **标签页管理**: 快速整理和查找打开的标签页。
- 🚀 **简洁易用**: 极简设计，开箱即用。

## 安装方法

1. 下载或克隆本仓库到本地。
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`。
3. 开启右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择本项目中的 `src` 目录。

## 开发

项目结构：
- `src/`: 核心源代码（Manifest V3）。
- `icons/`: 扩展图标。
